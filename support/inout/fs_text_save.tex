\section{Save Text to File}\label{save-text-to-file}}

Tabular outputs, text outputs, etc are saved as variables, which could
be printed in console. They can also be saved to file for future
re-used. For example, latex outputs need to be saved to file.

\begin{Shaded}
\begin{Highlighting}[]
\CommentTok{# Graphing Library}
\KeywordTok{library}\NormalTok{(kableExtra)}
\CommentTok{# Load Data}
\NormalTok{dt <-}\StringTok{ }\NormalTok{mtcars[}\DecValTok{1}\OperatorTok{:}\DecValTok{5}\NormalTok{, }\DecValTok{1}\OperatorTok{:}\DecValTok{6}\NormalTok{]}
\CommentTok{# Generate latex string variable}
\NormalTok{st_out_tex <-}\StringTok{ }\KeywordTok{kable}\NormalTok{(dt, }\StringTok{"latex"}\NormalTok{)}
\KeywordTok{print}\NormalTok{(st_out_tex)}
\end{Highlighting}
\end{Shaded}
